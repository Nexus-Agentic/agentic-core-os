# Contributing to Nexus-Agentic

Thank you for your interest in building the world's first native Agentic Operating System. We maintain a highly disciplined, enterprise-grade codebase.

## Code Quality Standards
1. **Memory Safety First:** All core kernel components must be written in pure, idiomatic Rust. The use of `unsafe` blocks must be meticulously documented, minimal, and explicitly justified.
2. **Real-World Execution:** We do not accept theoretical simulations. All submitted components must target real bare-metal architectures or verified low-level emulators (e.g., QEMU).
3. **Rewrites & Changes:** When submitting pull requests, do not leave trailing comments or partial modifications. Ensure that entire functional blocks or files are completely optimized and clean.

## Getting Started
1. Fork the repository under `Nexus-Agentic/agentic-core-os`.
2. Create your feature branch tracking our development roadmap.
3. Ensure all tests pass before opening a Pull Request.

For direct architectural inquiries, reach out to Tarvin or the maintainer core team.
